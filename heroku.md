# heroku

- heroku login: 登录
- heroku local web: 查看本地 localhost:5000
- heroku create (appName): 可自定义应用名 或者随机生成
- git push heroku master: 部署
- heroku open: 打开部署后的应用地址
- heroku logs --tail: 查看日志
